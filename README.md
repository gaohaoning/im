# im
java network im

## 1.程序源代码：
Network_Server: 服务器端程序（图像界面）
Network_Client: 客户端程序（图形界面）
Network_Client_Line: 客户端程序（命令行）

## 2.可执行程序：
Server.jar: 服务器端程序（图像界面）
Client.jar: 客户端程序（图形界面）
图形界面程序可以直接双击运行。
Client_Line.jar: 客户端程序（命令行）
客户端命令行程序需要执行批处理文件 Client_Line.bat 来执行

## 3.文档
设计文档和使用手册。
使用手册包括：
1.图形界面程序使用手册
2.命令行程序使用手册
